    3. Matplotlib
    4. Seaborn
    5. Scipy
    6. Scikit Learn
    7. Statsmodel
    8. Keras
    9. TensorFlow
    10. Pyspark
    
**Donde se abordaron modelos de...**
    
    1. Machine Learning
